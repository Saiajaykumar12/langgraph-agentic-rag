# LangGraph Agentic RAG System

An autonomous AI agent that retrieves 
information, reasons about it, and 
responds intelligently using LangGraph.

## What Makes This Different From Basic RAG
Basic RAG: Question → Retrieve → Answer
Agentic RAG: Question → Agent decides 
what to retrieve → Reasons → Acts → Answers

## Features
- Multi-step reasoning with LangGraph
- Autonomous web search using Google
- RAG with document retrieval
- QnA chatbot with memory

## Tech Stack
- LangGraph
- LangChain
- OpenAI GPT
- Groq LLaMA
- Python

## Setup
pip install -r requirements.txt
Add API keys in .env
Run agentic_rag.ipynb
